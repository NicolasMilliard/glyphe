# Glyphe

Glyphe is a registry-first, CSS-first toolkit for terminal-inspired text
animations on the web.

The project is inspired by copy-paste developer tooling: animations should be
easy to preview, customize, export, and own in your codebase.

## Stack

- Vite
- React
- TypeScript
- TanStack Router
- Tailwind CSS
- Bun

## Scripts

```sh
bun dev
bun run build
bun run lint
bun run format:check
```

## Direction

Glyphe will grow into a website, gallery, generator, registry, and later a CLI
for installable terminal-inspired animation primitives.
